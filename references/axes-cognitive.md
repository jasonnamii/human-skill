# 카너먼 7축 상세 — 내부 인과 체인 + 설계 원칙

## 축1. 인지 자원 경제학

**축 자명명제**: 인간의 인지 시스템은 유한한 자원을 최소 비용으로 운용한다

### 체인 1-① 자원 유한성 → S1 기본값 고정

```
주의력·작업기억은 용량 제한 자원
  → 인지 자원은 유한하며 사용할수록 고갈
    → 자원 고갈 시 소모 적은 대안 경로로 전환
      → S1은 자원 소모가 거의 없는 대안 경로
        → 에너지 최소화 생물학적 경향
          → 자연선택이 자동 처리를 기본값으로 고정
```

**의미**: 자원 유한성이라는 물리적 제약이 S1 우세를 필연으로 만든다. 선택이 아니라 구조적 귀결.

### 체인 1-② 자원 고갈 → 통제 실패 캐스케이드

```
자기 통제는 인지 자원 소모 과제
  → 연속적 통제는 자원을 고갈
    → 고갈 후 통제 실패 확률 증가
      → 자원 부족 시 S2 약화, S1 의존도 상승
        → 자원 충분 시에도 S2 가동 지연
```

**의미**: 통제력은 사용할수록 약해지는 소모성 자원. "의지가 약하다"가 아니라 "자원이 고갈됐다"가 정확한 기술.

### 체인 1-③ 인지 부하 → 속성 대체 경로 활성

```
어려운 질문은 높은 인지 부하
  → 부하 높으면 S1이 대안을 자동 탐색
    → 유사성 판단은 자동·즉각적(S1)
      → S1이 먼저 답 제시하면 S2는 검증 없이 수용
```

**의미**: 인지 부하가 임계점을 넘으면 속성 대체가 자동 기동. 축1(자원경제)이 축2(속성대체)를 구동하는 인과 경로.

### 설계 원칙

- 중요 판단은 고자원 타이밍 배치
- 자동 처리로 인지 부하 최소화
- 피로 상태 판단 회피
- M1 연동: 자원 유한→감정이 기본 판단 모드(축5)

## 축2. 속성 대체와 용이성

**축 자명명제**: 인간은 어려운 판단을 쉬운 속성으로 자동 대체하며, 처리 용이성을 진실·중요성의 지표로 사용한다

### 체인 2-① 속성 대체 메커니즘

```
대상 속성과 연관된 더 쉬운 속성이 존재
  → 전형적 개체(프로토타입)는 자동 인출
    → "집합의 속성"이 "전형의 속성"으로 대체
      → 확률이 유사성으로 대체
        → 유사성과 확률 구분 실패
          → 대체가 일어났다는 사실 자체를 의식하지 못함
```

**의미**: 대체는 3중으로 은폐 — 자동 실행(S1), 원천 혼동(구분 실패), 메타인지 부재(비자각). 교정이 극도로 어려운 이유.

### 체인 2-② 용이성 → 진실 환상

```
반복 노출은 처리 용이성 증가
  → 처리 용이성은 메타인지 신호로 경험
    → 용이성의 원천(반복, 글꼴, 기분)을 구분 못함
      → 용이성이 "안전/친숙/진실"로 자동 해석
        → 이 해석이 내용의 실제 진위와 독립적
```

**의미**: "익숙하다"와 "사실이다"가 신경 수준에서 동일한 신호를 공유. 반복만으로 진실을 만들 수 있다.

### 체인 2-③ 용이성 ↔ 긴장 쌍극

```
용이성 → "안전/친숙/진실"
  ↕ (거울상)
인지 긴장 → "무언가 잘못되었다" 경보
  → 경보가 S2를 활성화
    → S2 활성화로 신중·분석적 처리
```

**의미**: 용이성과 긴장은 동일 스펙트럼의 양극. 의도적으로 긴장을 유발하면(어려운 글꼴, 낯선 형식) S2 활성화를 강제 — 넛지의 메커니즘적 기반.

### 설계 원칙

- 신뢰 필요 시 용이성 강화 (단순 언어, 친숙 프레임)
- 분석 강제 시 긴장 활성화 (대조, 난이도 상향)
- M3 연동: 속성 대체가 스토리텔링(축4)의 왜곡된 재료 공급

## 축3. 손실 회피와 참조점

**축 자명명제**: 인간은 참조점 대비 변화로 가치를 평가하며, 손실을 이득보다 강하게 느낀다

### 체인 3-① 감각 원리의 가치 동형 적용

```
감각 시스템은 자극의 변화에 반응(Weber-Fechner)
  → 변화 감지 원리가 가치 판단에 동일 적용
    → 절대값이 아닌 참조점 대비 변화로 가치를 느낌
      → 참조점은 현재 상태·기대·사회적 비교에 의해 설정
        → 동일 최종 상태도 참조점 다르면 다른 가치
```

**의미**: 가치 함수는 감각 시스템의 확장. "좋다/나쁘다"는 절대적 상태가 아니라 참조점과의 거리.

### 체인 3-② 손실 비대칭 → 4대 파생 현상

```
손실은 위협 신호 → 부정 감정 즉각 활성
  → 이득의 긍정 감정 < 동일 크기 부정 감정
    → 진화적 고정(위협 과대평가가 적응적)
```

이 비대칭이 4갈래로 파생:

```
손실 회피 → 소유 효과(포기=손실)
         → 현상 유지(변화의 단점 > 장점)
         → 매몰 비용(확정 손실 회피 → 비합리적 지속)
         → 근시안적 회피(짧은 주기에서 손실 회피 강화)
```

**의미**: 단일 원인(손실 회피) → 다중 현상(소유 효과·현상 유지·매몰 비용·근시안적 회피)의 전형.

### 체인 3-③ 가치 함수의 곡률 → 위험 태도 반전

```
자극 강도 증가 시 감지 역치 상승
  → 이득 영역: 오목(한계 가치 감소) → 위험 회피
  → 손실 영역: 볼록(한계 고통 감소) → 위험 추구
```

**의미**: 같은 사람이 이득 앞에서는 보수적이고 손실 앞에서는 도박꾼이 된다. 인격의 불일치가 아니라 가치 함수의 곡률 전환.

### 설계 원칙

- 참조점을 의도적 설정하여 지각된 가치 조작
- 손실 프레이밍 (효과 2.5배)
- 소유감 선행으로 이탈 비용 증가
- EM1 연동: 손실 회피는 오류 관리의 특수 사례 (궁극적 원인=축G)

## 축4. 자동 스토리텔링과 과신

**축 자명명제**: 인간은 가용 정보로 일관된 인과 스토리를 자동 구성하며, 그 일관성을 확신과 동일시한다

### 체인 4-① WYSIATI → 과신 캐스케이드

```
S1은 가용 정보로 일관된 인과 스토리를 자동 구성
  → 정보 누락 사실 자체가 인식 불가
    → 스토리 일관성이 높을수록 확신 상승
      → 확신은 정보의 양이 아닌 일관성에 의해 결정
        → WYSIATI: 자신이 아는 것이 전체로 느껴짐
          → 통제 환상: 결과에 대한 영향력 과대평가
            → 과신이 통제력·역량 과대평가 유발
```

**의미**: 과신은 정보 과잉이 아니라 정보 부재의 비자각에서 온다. 모르는 것을 모르기 때문에 확신한다.

### 체인 4-② 사후 확신 강화 루프

```
S1은 인과적 설명을 자동 구성
  → 결과를 알면 기억 속 예측이 결과 방향으로 수정
    → 수정된 기억이 결과와 일치 → "예견 가능" 확신
      → 사후 확신이 스토리 설득력 강화
        → WYSIATI에 의해 유일한 설명으로 느껴짐
          ↻ (피드백 루프: 과신 강화)
```

**의미**: 과신은 자기강화 루프. 사후에 "역시 알고 있었다"는 기억 왜곡이 미래의 과신을 정당화.

### 체인 4-③ 전문가 직관의 유효 조건

```
직관은 반복 경험에서 학습된 패턴의 자동 인식
  → 환경의 규칙성 필요 + 충분한 반복 필요 + 즉각 피드백 필요
    → 하나라도 결여 시 무작위 추측과 구분 불가
      → 저유효성 환경에서 패턴은 노이즈 환상
        → 피드백 부재·지연으로 환상 비교정
```

**의미**: 전문가 직관이 유효한 영역은 극도로 제한적(소방관, 체스). 대부분의 전문가 판단(주식, 정치, 채용)은 타당성 환상.

### 설계 원칙

- 불일치 정보로 과신 차단
- 사전 예측 기록·공개 → 추후 검증 강제
- 전문가 직관 전 유효 조건 검증
- EM3 연동: 과신은 자기기만을 통한 번식 전략 (궁극적 원인=축F)

## 축5. 감정 선행과 프레이밍

**축 자명명제**: 감정 반응이 판단에 선행하며, 정보의 표상·전달 방식이 감정을 통해 판단을 결정한다

### 체인 5-① 감정 선행 → 판단 결정

```
감정적 반응은 대상 접촉 시 자동·즉각 발생
  → 감정 긍정이면 이익↑ 위험↓ 추정(역상관 생성)
    → 역상관은 실제 세계에 없지만 감정적 일관성이 생성
      → 단일 평가 시 S1 감정 반응이 유일 근거
```

**의미**: 감정은 판단의 "오염원"이 아니라 판단 그 자체의 기본 모드. S2가 개입하지 않으면 감정=판단.

### 체인 5-② 프라이밍 → 프레이밍 → 선택 결정

```
자극 노출 → 관련 개념·감정·행동 자동 활성(확산 활성화)
  → 활성화된 개념이 후속 정보 처리의 필터로 작동
    → 선택은 정보의 객관적 내용 + 표상 방식에 의존
      → 표상 방식이 다르면 다른 연상·감정 활성
        → S1은 논리적 동치가 아닌 표면 형식에 반응
```

**의미**: 동일 정보가 "90% 생존"과 "10% 사망"에서 다른 선택을 유발. 정보의 내용이 아니라 포장이 행동을 결정.

### 체인 5-③ 진화적 감정 고정

```
위협 과대평가가 적응적 → 진화적 고정
  → 공정성 강제의 사회적 협력 기능
    → 공정성 위반 → 부정 감정 즉각 활성
      → 감정 강도 > 경제적 손실 고통
        → 처벌 동기가 자기 이익을 능가
```

**의미**: 공정성 감각은 학습이 아니라 진화적 장치. 비합리적으로 보이는 이타적 처벌이 집단 수준에서 협력을 유지.

### 설계 원칙

- 감정 상태를 먼저 세팅 후 정보 제공
- 손실 프레이밍 활용 (심리 활성화)
- 공정성은 설계 불가 (읽기만)
- EM2 연동: 감정 선행은 다시스템 조율의 진화적 필연 (궁극적 원인=축G)

## 축6. 기억 편집과 경험 왜곡

**축 자명명제**: 기억은 경험을 선별·왜곡 저장하며, 기억된 버전이 미래 의사결정을 지배한다

### 체인 6-① 경험 소멸 → 기억 지배

```
매 순간의 경험은 대부분 기록되지 않고 소멸
  → 기억은 모든 순간을 동등하게 저장하지 않음(비균등 인코딩)
    → 감정 강도 높은 순간(peak)이 과대 표상
    → 마지막 순간(end)이 과대 표상(최신성 효과)
      → 회고 평가 = 과대 표상된 순간들의 평균
        → 의사결정은 경험이 아닌 기억에 기반
```

**의미**: 경험하는 자아와 기억하는 자아는 다른 이해관계를 갖는다. 의사결정을 지배하는 것은 기억하는 자아이며, 이 자아는 경험을 왜곡한다.

### 체인 6-② 집중 환상 → 감정 예측 실패

```
주의 대상이 인지적 전경으로 부상
  → 전경의 인지적 무게 과대
    → "지금 생각 중" = "중요하다" 자동 전환
      → 미래 감정 예측 시 집중 환상 작동
        → 해당 사건만 집중, 공존 요인 무시
          → 심리적 면역 체계의 감정 감소 비반영
            → 적응 능력 비반영 → 감정 예측 오류
```

**의미**: 승진·결혼·이사 같은 큰 변화의 행복 효과를 과대추정하는 이유. 주의가 집중되는 순간에만 중요하게 느껴지고, 적응은 예측에 반영되지 않는다.

### 설계 원칙

- 피크 순간 의도적 설계
- 마지막을 긍정적으로 마무리
- 기간보다 피크+엔드 중시
- 감정 예측 시 적응 보정
- M5 연동: 감정→기억→감정의 자기강화 루프

## 축7. 판단 잡음

**축 자명명제**: 동일 판단에 대해 사람 간·시점 간 체계적 비일관성이 존재하며, 이를 자각하지 못한다

### 체인 7-① 3대 잡음원의 독립 작동

```
개인마다 성격·경험·가치관 상이
  → 개인차가 판단의 전반적 기준선 결정 (수준 잡음)

동일 사안의 측면별 민감도가 개인마다 상이
  → 경험·전문성·가치 가중치가 민감도 결정 (패턴 잡음)

판단 시점의 내적 상태가 인지 처리에 영향
  → 외부 환경이 내적 상태를 변화
    → 동일인이 시점별로 다른 판단 (기회 잡음)
```

세 잡음원의 공통: 기준선 차이·내적 영향 모두 자각되지 않음 → 잡음의 크기를 체계적으로 과소평가.

**의미**: 편향(bias)은 방향이 있어 식별·교정 가능하지만, 잡음(noise)은 방향이 없어 보이지 않는다. 편향+잡음=판단 총오류(M6).

### 설계 원칙

- 잡음은 개인 반성·교육으로 제거 불가
- 프로세스 표준화·체크리스트·중위수 집계
- 판단자 자율성보다 프로세스 일관성 우선
- 규칙을 사전에 명확히 설정

  ↓
