# 행동설계 고유축I 상세 — 내부 인과 체인 + 설계 원칙

## 고유축I. 행동 루프와 습관 자동화 역학

**축 자명명제**: 인간의 행동은 트리거-갈망-반응-보상의 4단계 루프로 형성되며, 반복에 의해 기저핵이 인수하여 자동화되고, 자동화된 루프는 삭제 불가하며 대체만 가능하다

---

### 체인 I-① 행동 발생→루프 형성→자동화 (형성 체인)

```
마찰-동기 임계(행동선)
  │ 프롬프트가 임계 이상에서 행동 발화
  ↓
단발 행동 실행
  │ 보상 경험 (즉각적 긍정 감정 or 부정 감정 해소)
  ↓
단서-보상 연합 형성 (고전적 조건화)
  │ 반복에 의한 연합 강화
  ↓
갈망 발생 (단서 시점 도파민 발화)
  │ 갈망이 루틴 실행을 추동
  ↓
습관 루프 확립 (단서→갈망→루틴→보상)
  │ 충분한 반복
  ↓
기저핵 청킹 (피질→기저핵 전환)
  │ 의식적 통제에서 자동 실행으로
  ↓
자동화 완성 (인지 자원 해방)
```

**핵심 원리**: 행동은 MAP 임계를 넘어 발생→보상에 의해 강화→반복에 의해 연합 누적→임계 반복 후 기저핵이 인수하여 자동화. 이 체인은 비가역적 — 자동화된 루프는 삭제 불가, 대체만 가능.

---

### 체인 I-② 루프 자기강화→투자→고착화 (강화 체인)

```
습관 루프 확립
  │ 각 사이클이 투자를 유발
  ↓
투자에 의한 저장 가치 축적
  │ ❶미래 보상 질 향상 + ❷다음 트리거 자동 로딩
  ↓
보상 질 향상 → 갈망 강화
  │ + 트리거 빈도 증가
  ↓
사이클 빈도 증가 + 강도 증가
  │ 양의 피드백 루프
  ↓
외부→내부 트리거 전환
  │ 부정 감정이 자동 트리거로 작동
  ↓
외부 의존 탈피 → 자기영속적 루프
  │ + 전환 비용 증가 (매몰비용 축적)
  ↓
고착화 (습관 또는 중독)
```

**핵심 원리**: 루프의 각 반복이 다음 반복을 강화하는 양의 피드백 구조. 투자→가치→갈망→빈도→투자의 자기증폭이 외부 트리거 의존을 탈피시키고, 전환 비용을 축적하여 이탈을 차단. 건강한 습관에도 중독에도 동일하게 작동 — 방향은 루프 내용이 결정.

**가변 보상의 위험**: 보상 크기/시점 불확실→도파민 지속적 상승 유지. 고정 보상은 적응하나 가변 보상은 적응 불가. 슬롯머신·SNS 좋아요·게임 리워드의 중독 기제.

---

### 체인 I-③ 복리 축적→돌파→숙달 (성장 체인)

```
타이니 해빗 / 2분 규칙 (최소 행동)
  │ 동기 의존성 제거, 실행 보장
  ↓
반복에 의한 미세 개선 축적 (1% 복리)
  │ 초기: 잠재력 고원 (가시적 결과 없음)
  ↓
실망의 계곡
  │ 선형 기대 vs 비선형 현실 → 포기 압력
  │ 즉각적 보상(Shine) + 정체성 기반 동기가 다리 역할
  ↓
임계점(breakthrough) 돌파
  │ 축적된 잠재력이 역치 초과
  ↓
가시적 성과 + 자기효능감 급증
  │ 핵심 습관→작은 승리→연쇄 변화
  ↓
숙달 단계 진입
  │ 자동화 완성 → 지루함 위협
  ↓
골디락스 규칙에 의한 적절 도전 → 몰입
  │ 의도적 연습으로 전환
  ↓
장기 유지와 지속적 성장
```

**핵심 원리**: 행동의 효과는 비선형(복리)으로 축적. 초기 구간(잠재력 고원)에서 포기를 방지하는 것이 핵심 설계 과제. 돌파 후에는 자동화→지루함→적절 도전→몰입의 2단계 전략으로 전환.

## 허브 축 특성 — 8개 메타원리 접합

고유축I는 8개 축과 직접 교차하는 **실행 허브 축**이다.

| 메타원리 | 접합 | 방향 | 핵심 |
|---------|------|------|------|
| BM1 | 축I↔축1 | 쌍방향 | 습관 자동화→인지 자원 해방 / 자원 고갈→습관 퇴행 |
| BM2 | 축5→축I | 단방향 | 감정이 트리거(부정 감정)와 보상(긍정 감정)을 동시 공급 |
| BM3 | 축I↔축B | 쌍방향 | 습관→정체성 증거 축적, 정체성→동기 비용 제거 (자기강화 루프) |
| BM4 | 축I↔축E | 쌍방향 | 보상→욕구 충족 경로 결정, 욕구 충족→루프 에너지 공급 |
| BM5 | 축I↔축D | 쌍방향 | 자율적 선택→의지력 보호, 자동화→자율성 경험 강화 |
| BM6 | 축3→축I | 단방향 | 손실 회피→기존 습관 고착 + 새 습관 포기 동시 유발 |
| BM7 | 축H→축I | 단방향 | 감정 조절 실패→습관 루프를 비상 출구로 활성화→습관 의존 증가 |
| BM8 | 축G→축I | 단방향 | 환경 불일치→현대 초과 자극과 결합→습관 루프 부적응적 과잉 강화 |

## 마찰-동기 임계선

| 목표 | 마찰 전략 | 사례 |
|------|----------|------|
| 원하는 행동 촉진 | 마찰 감소 | 운동복을 침대 옆에 배치 / 서브스크립션 자동갱신 |
| 원치 않는 행동 억제 | 마찰 증가 | SNS 앱 삭제 / 신용카드를 다른 방에 보관 |

마찰 > 동기 → 행동 억제. 동기 > 마찰 → 행동 실행.

## 설계 매트릭스

| 목표 | 설계 전략 | 핵심 체인 | 위험 요소 |
|------|----------|---------|---------|
| 새 습관 형성 | 마찰↓ + 기존 루틴 편입 + 즉시 보상 | I-① | 마찰이 높으면 실패 |
| 나쁜 습관 제거 | 트리거 회피 + 마찰↑ + 대체 보상 | I-② 역 | 트리거 제거 실패 시 재발 |
| 습관→정체성 | 반복→증거 축적→선언 | I-②→I-③ | 부정적 정체성도 동일 루프로 강화 |
| 숙달 달성 | 복리 축적 + "실망의 계곡" 숙지 | I-③ | 초기 진행 미감지로 포기 |
| 중독 예방 | 가변 보상 제한 + 고정 보상 전환 | I-② 감시 | 무의식적 강화 회피 어려움 |
