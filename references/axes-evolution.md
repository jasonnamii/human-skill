# 진화심리학 고유 2축 상세 — 내부 인과 체인 + 설계 원칙

## 고유축F. 번식 전략과 짝짓기 역학

**축 자명명제**: 인간은 성별에 따라 비대칭적인 부모 투자 구조를 가지며, 이에 따라 짝 선택 기준·짝짓기 전략·짝 보호·질투 메커니즘이 체계적으로 분화한다

### 체인 F-① 부모 투자 비대칭 → 성별 분화된 짝 선택 기준

```
인간 여성은 임신·수유로 자녀 당 생물학적 투자가 크다
  → 더 많이 투자하는 성이 짝 선택에서 더 신중해야 한다
    → 여성의 큰 투자 → 짝 선택 실패 비용이 극대
      → 자원·지위 높은 짝 = 자녀 생존 확률 증가
        → 헌신 능력 = 자원 집중 투자 신뢰
          → 남성은 유전자 전달이 주요 기여 → 번식력 단서 선호
```

**의미**: 부모 투자의 비대칭이 짝 선택 기준의 성별 분화를 생성. 여성(자원·헌신)과 남성(젊음·건강)은 각각 자녀 투자 확보와 유전자 질 확보라는 상이한 적응 문제의 해결책. 보편적(~70%)이나 개인차·문화 강도 크다.

### 체인 F-② 짝 가치 평가 → 전략 전환 → 성별 단기 전략 분화

```
자신의 짝 가치가 높으면 높은 짝을 기대, 낮으면 기대 감소
  → 자원 풍부→다수 짝 가능, 부족→안정적 짝 필수
    → 조건 변화 → 장기(자원 투자)/단기(다양성) 전략 전환
      → 남성 단기: 번식 비용 낮음 → 유전자 광포 전략
      → 여성 단기: 투자 비용 큼 → 유전자 질/자원 추출
```

**의미**: 짝짓기 전략은 고정이 아니라 자기 짝 가치와 자원 상황에 따른 유연한 전환. 남녀의 단기 전략이 다른 것은 투자 비대칭에서 논리적으로 도출.

### 체인 F-③ 부성 불확실성 → 질투·짝 보호 → 중복 방어 체계

```
남성은 은폐된 배란과 외부 짝짓기 가능성으로 부성 불확실
  → 부성 불확실 → 타인 자녀에 대한 투자 위험 → 적응도 손실
    → 성적 불륜 = 부성 불확실성 직접 생성 → 남성의 최대 위협
      → 이익 제공(자원·외모 과시) → 짝의 이탈 동기 감소
      → 비용 부과(경쟁자 위협·감시) → 이탈 비용 증가
        → 질투·감시·처녀성 선호 = 중복 방어 체계
```

**의미**: 부성 불확실성이라는 단일 적응 문제가 성적 질투·짝 감시·처녀성 선호라는 다층 방어 체계를 생성. 방어의 중복성은 문제의 적응적 중요도를 반영.

### 설계 원칙

- 신호 자동탐지: MV 신호(지위, 신체, 자원)는 자동 탐지됨. 신호 조작은 단기만 효과
- 선호도 성별차: 보편적이나 개인차·문화 강도 크다. 성별 고정관념으로 개인 예측 금지
- 질투 메커니즘: 설계 대상 아님 (자동, 고정, "읽기 전용")
- EM3 연동: 과신은 자기기만을 통한 번식 전략 (지위 획득·짝 유인에서 적응적)
- EM8 연동: 번식 선택압이 지위 추구를 내재적 동기로 설계

## 고유축G. 위협 관리와 환경 불일치

**축 자명명제**: 인간은 조상 환경(EEA)의 위협에 최적화된 방어·혐오·공포 체계를 가지며, 이 체계가 현대 환경과 불일치하여 체계적인 부적응 반응을 생성한다

### 체인 G-① 반복적 적응 문제 → 영역 특이적 설계 → 트리거 활성화

```
진화적 역사 동안 인간은 반복적으로 동일한 환경 문제에 노출
  → 자연선택은 특정 문제 해결에 효과적인 심리 장치를 보존
    → 일반 체계는 각 문제에 최적화되지 않음
      → 영역 특이적 장치 = 해당 문제에 효율성 최고
        → 특정 환경 신호(트리거)에 대응하도록 설계
          → 에너지 절약 → 트리거 없으면 휴면
```

**의미**: 영역 특이성과 트리거 활성화는 진화 설계의 두 가지 핵심 원리. "왜 일반적 지능이 아니라 특수 모듈인가"에 대한 답: 에너지 효율성과 문제 해결 정확도의 이중 이점.

### 체인 G-② 조상 위협 준비 → 행동 면역 → 3영역 혐오 분화

```
조상 환경의 위협은 수십만 년간 생존을 위협
  → 신경 회로를 사전에 부분적으로 구성(준비된 학습)
    → 감염은 물리적 접촉을 통해 발생
      → 혐오 감정 = 오염원 자동 회피 동기 시스템
        → 병원체 혐오: 부패·오염·체액
        → 성적 혐오: 근친·부적절 짝(번식 비용)
        → 도덕적 혐오: 규범 위반·무임승차(집단 위협)
```

**의미**: 혐오는 단일 감정이 아니라 3영역으로 분화된 방어 체계. 병원체→성적→도덕적으로의 확장은 물리적 오염 회피에서 사회적 오염 회피로의 진화적 전이(exaptation).

### 체인 G-③ EEA 설계 + 현대 환경 = 체계적 부적응

```
심리 메커니즘은 수십만 년 조상 환경(EEA)에서 형성
  → 현대 환경은 EEA와 구조적으로 다름
    → EEA 최적화 메커니즘 + 현대 신호 = 설계 의도와 다른 결과
      → 칼로리 선호 + 과잉 환경 → 비만
      → 소규모 사회 인지 + 대규모 익명 사회 → 사회적 불안
      → 제한 비교 메커니즘 + 무한 비교(SNS) → 만성 박탈감
      → 제한 성적 경쟁 + 무한 경쟁(데이팅앱) → 질투 만성화
```

**의미**: 환경 불일치는 단일 현상이 아니라 체계적 패턴. 4개 영역 모두 동일한 구조(EEA 적응 + 현대 초과)를 공유. "메커니즘의 오류가 아니라 환경의 불일치가 원인"이라는 프레임이 현대 부적응 진단의 기본 도구.

### 설계 원칙

- 위협 모듈 고정성: 설계 불가 (자동, 생물학적 고정)
- 트리거 환경 설계: 신호 환경을 제어 가능 (뉴스 다이어트, SNS 필터링)
- 혐오 메커니즘: 분석은 가능, 조작은 불가. 도덕적 혐오로 편견을 "정당화"하지 말 것
- EM1 연동: 손실 회피(축3)는 오류 관리의 특수 사례
- EM2 연동: 감정 선행(축5)은 다시스템 조율의 진화적 필연
- EM7 연동: 환경 불일치가 축1·3·5·6의 부적응을 체계적으로 생성

## 진화축의 특수한 위치

진화축(F·G)은 다른 축과 다르다: **근접적 메커니즘이 아니라 궁극적 원인**을 다룬다.

| 구분 | 근접 축 (1~7, A~E, H, I) | 진화 축 (F, G) |
|------|------------------------|---------------|
| 질문 | "어떻게 작동하는가?" | "왜 이 메커니즘이 존재하는가?" |
| 설계 대상 | 직접 조작 가능 | 메커니즘 자체는 "읽기 전용" |
| 설계 방향 | 트리거·환경·프레이밍 | 환경 설계로 간접 영향 |
| 진단 가치 | 현상 설명 | 현상의 근본 원인 설명 |

**설계 레버는 근접 축(1~7, A~E, H, I)에 있다. 진화축은 "왜"를 설명하여 근접 축의 설계를 안내한다.**
