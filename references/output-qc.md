# 출력 QC 체크리스트

> Human Skill 스포크. 모든 모드에서 산출물 제출 전 검증 필수.
> 시뮬레이션에서 M3 복사오류 같은 문제 재발 방지를 위한 구조적 안전장치.

---

## 공통 4항목

| # | 검증 항목 | 검증 방법 | 실패 시 |
|---|---|---|---|
| ①축 활성 근거 | 각 활성 축에 대해 "이 축이 활성인 근거" 1줄 역추적 가능한가? | 근거 없는 축 → 비활성으로 재판정 | 해당 축 제거 |
| ②메타원리 방향성 | 사용된 메타원리의 방향(→ 단방향 / ↔ 쌍방향)이 정확한가? | `--detail`로 방향 확인 | 방향 수정 |
| ③사례간 중복 | 복수 사례 분석 시 동일 내용이 다른 사례에 복사되지 않았는가? | 사례별 고유 활성축 조합 확인 | 중복 사례 재분석 |
| ④허브축 연결 | 활성 축 3개+ 시 허브축(H·I·E)과의 연결이 최소 1개 식별되었는가? | `--hubs`로 교차 확인 | 허브 연결 추가 탐색 |

## 예측 모드 추가

①~④ + ⑤확률 근거 태그(§5) 부착 여부 검증.
