# 치알디니 고유 3축 상세 — 내부 인과 체인 + 설계 원칙

## 고유축A. 사회적 의무감과 호혜

**축 자명명제**: 인간은 사회적 교환에서 자동적 부채감과 의무감을 경험하며, 이 감정이 합리적 판단을 압도한다

### 체인 A-① 호의 수령 → 자동 부채감 생성

```
호의는 대뇌 보상 회로를 활성화
  → 보상 회로 활성화가 부채 상태 신호 생성
    → 부채감은 의도 여부 무관, 호의 감지 시 자동 생성
      → 의식적 통제 없이 자동 발생
        → 원시적 뇌 영역에 위치하여 맥락 무시
```

**의미**: 부채감은 선택이 아니라 신경학적 자동 반응. 호의의 의도성도 맥락의 합리성도 이 자동 반응을 차단하지 못한다.

### 체인 A-② 부채감 → 수용 강제 캐스케이드

```
부채감은 심리적 불편함(불안정 상태) 생성
  → 불편함의 해소 수단은 빚을 갚는 행동뿐
    → 상대 요청 거절 시 불편함 증가
      → 맥락적 합리성과 감정 충돌 시 감정 우선
        → 심리적 불편함 회피가 수용을 강제
```

**의미**: 부채감은 해소될 때까지 심리적 압력을 지속하며, 거절 시 압력이 증폭. 상호성 원칙의 설득력이 극단적으로 높은 구조적 이유.

### 체인 A-③ 양보 크기 자동 보정 메커니즘

```
타인의 양보는 사회적 규범 신호 생성
  → 뇌는 양보 크기를 자동 추적
    → 상대 양보 크기가 공정성 판단의 기준
      → 공정성 기준 불일치는 불편감 생성
        → 불편감 해소를 위해 자신의 양보 크기 자동 조정
```

**의미**: 호혜에는 내장된 비례 보정 시스템 존재. 큰 양보→큰 역양보로 자동 조정되므로 door-in-the-face 전략이 작동.

### 설계 원칙

- 호의 선행: 먼저 가치를 제공하라
- 양보 크기 전략화: 큰 요청→거절→작은 요청 단계화
- 부채감 자동성: 맥락 독립, 무의식적 작동 — 교정 극히 어려움
- CM1 연동: 인지 자원 고갈 시 부채감의 무비판적 수용이 증폭

## 고유축B. 자기상 일관성과 몰입 에스컬레이션

**축 자명명제**: 인간은 자기상을 핵심 자원으로 보호하며, 과거 행동·선언과의 일관성을 유지하기 위해 비합리적 몰입을 자동으로 확대한다

### 체인 B-① 행동 → 자기상 형성 → 일관성 강제

```
자기상은 자아 정체성의 핵심이며 뇌의 지속적 우선순위
  → 작은 약속 충족 시 일관성 기반 자기상 형성
    → 새로운 선택은 자기상과의 일치성이 가장 높은 옵션으로 수렴
      → 초기 약속이 자기상의 핵심 요소가 됨
        → 상황 변화보다 자기상 유지가 높은 심리적 우선순위
```

**의미**: 한번 형성된 자기상은 후속 선택을 일관성 방향으로 강제. foot-in-the-door 전략의 인과적 기반.

### 체인 B-② 인지 부조화 → 태도 자동 변화

```
불일치는 인지 부조화 → 심리적 불편함 유발
  → 뇌는 통증 해소를 위해 자동으로 인지적 왜곡 수행
    → 태도 변화가 가장 효율적인 왜곡
      → 의식과 무관하게 태도는 선택과 일치하도록 자동 조정
```

**의미**: 행동이 먼저, 태도가 나중. 인간은 행동을 바꾸는 대신 태도를 바꿔서 부조화를 해소. 몰입 에스컬레이션의 엔진.

### 체인 B-③ 비용 투자 → 이탈 불가 고착

```
높은 노력 비용은 높은 가치 신호
  → 주의 강화 → 자기상 투자 증가
    → 자기상 투자가 높을수록 포기 시 손상 증가
      → 입장 변화는 자기상 약화와 평판 손상을 동시에 의미
        → 자기상 약화 회피가 입장 고수를 강제
```

**의미**: 투입한 비용 자체가 이탈을 차단하는 심리적 장벽. 카너먼의 매몰비용(축3, 인지적)과 달리 여기서는 자기상 보호라는 동기적 힘이 작동.

### 설계 원칙

- Foot-in-the-door: 작은 동의로 시작→자기상 형성→큰 동의 확장
- 공개 선언: 자기상 고정의 강화 장치
- 매몰비용 이중 고착: 투자된 비용(축3) + 자기상 위협(축B) 동시 작동
- CM2 연동: 손실 회피가 자기상 보호를 증폭하여 몰입 에스컬레이션 트랩 형성

## 고유축C. 정체성 통합과 내집단 자동 이타성

**축 자명명제**: 인간은 공유 정체성을 감지하면 상대를 자기의 확장으로 처리하며, 무조건적 돌봄과 이타성이 자동 활성화된다

### 체인 C-① 정체성 일치 → 자기 확장 → 무조건적 이타

```
정체성 일치는 거울 뉴런 강력 활성화
  → 강력한 거울 뉴런은 상대를 신체적 자아의 확장으로 처리
    → 자신의 신체 확장은 자동으로 돌봄·이타성의 대상
      → 정체성 통합이 무조건적 이타성 생성
        → 생존 본능의 최고 우선순위가 의식적 판단을 압도
```

**의미**: 정체성 통합은 비용-편익 계산을 우회. "너는 나다"라는 존재론적 동일시가 발생하면, 의식적 이해관계 판단 없이 이타 행동이 자동 실행.

### 체인 C-② 공유 경험 → 정체성 통합 촉진

```
공동 경험과 환경은 집단 정체성의 신호
  → 공동 고생은 진화적 생존 협력 신호
    → 생존 협력은 신뢰·동지애 자동 유발
      → 집단 정체성은 내집단-외집단 구분 자동 생성
        → 내집단 인식은 자동으로 신뢰도·호감 증가
```

**의미**: 정체성 통합은 혈연만이 아니다. 공동 경험(고생, 성공, 식사)이 대리 신호로 작동. 통일성(Unity)과 호감(Liking)의 차이 — 호감은 유사성, 통일성은 동일성.

### 체인 C-③ 신체적 동기화 → 화학적 결합

```
신체 동작 모방은 거울 뉴런 활성화
  → 거울 뉴런은 신체적 동기화 생성
    → 공동 식사는 화학적 동기화 유발
      → 신체 접촉은 옥시토신 분비 자동 유발
        → 옥시토신은 신뢰·친밀감의 신경 생화학적 기초
```

**의미**: 정체성 통합에는 하향(개념적 동일시)과 상향(신체적 동기화) 두 경로 존재. 상향 경로는 옥시토신이라는 생화학적 메커니즘을 통해 작동.

### 설계 원칙

- 공유 경험 설계: 고통이든 기쁨이든 함께하기
- 동시 행동(동기화): 행진·노래·식사 등 유대 자동 생성
- 정체성 통합은 가장 강력하나 비가역성 주의
- EM6 연동: 행동 면역의 "비익숙=오염" 휴리스틱이 외집단 편견의 생물학적 기원

## 고유 축간 연결

### CM7: 고유축A ↔ 고유축B
호혜 행동이 자기상을 형성하고("나는 보답하는 사람"), 형성된 자기상이 호혜를 지속시킨다. 외부 의무가 내면적 정체성으로 전환되는 루프. door-in-the-face(축A)→foot-in-the-door(축B)로의 전환 경로.

### CM8: 고유축C ↔ 고유축B
정체성 통합이 자기상의 경계를 확장시키고("우리 집단 사람"), 확장된 자기상이 집단 이탈을 차단한다. "나"에서 "우리"로의 불가역적 확장이 몰입 에스컬레이션을 집단 수준으로 증폭.

## 3축 공통 구조

| 축 | 감지 | 감정 압력 | 해소 행동 |
|---|---|---|---|
| 고유축A | 호의 감지 | 부채감(불편함) | 보답 행동 |
| 고유축B | 불일치 감지 | 인지 부조화(통증) | 태도 변화/몰입 |
| 고유축C | 정체성 일치 감지 | 돌봄 충동(옥시토신) | 이타 행동 |

**트리거 우선순위**: C > B > A (깊이: 정체성 > 자기상 > 부채감)
