# 44개 메타원리 레퍼런스 맵

44개 메타원리는 16축 간의 상호작용을 정의합니다. 각 원리는 **축의 접합(방향) → 메커니즘 → 설계 함의**로 구성.

---

## 1. 카너먼 6 메타원리 (M1~M6)

| # | 접합 | 메커니즘 | 설계 함의 |
|---|---|---|---|
| M1 | 축1→축5 | 인지 자원 유한 → 감정이 기본 판단 모드. S2는 예외적 | 고부하 상황에서 감정 경로만 작동 |
| M2 | 축3↔축5 | 손실 고통(2.5배) + 감정 증폭 → 변화 거부 자동 출력 | 손실 프레이밍 회피, 참조점 선이동 |
| M3 | 축2→축4 | 속성 대체 왜곡 → 일관성 스토리 → 현실 괴리 | 스토리 일관성 ↑ = 입력 왜곡 의심 ↑ |
| M4 | 축4↔축6 | 현재 스토리↔과거 기억, 미래 예측 양방향 편집 | 사전 예측 기록, 기억 편집 인식 |
| M5 | 축5↔축6 | 감정 필터→기억 회수→감정 강화(자기강화 루프) | 부정 감정 루프 차단: 감정 상태에서 의사결정 보류 |
| M6 | 축4 독립 | 편향(방향) + 잡음(분산) = 총오류. 독립적 오류원 | 편향 제거만으로 부족. 잡음 감소 별도 필요 |

## 2. 치알디니 8 메타원리 (CM1~CM8)

| # | 접합 | 메커니즘 | 설계 함의 |
|---|---|---|---|
| CM1 | 축1→축A | 인지 절약: 호혜 의무감 무비판 실행 자동화 | 고부하 시 호혜 요청 더 효과적 |
| CM2 | 축3↔축B | 손실 회피 + 자기상 포기 고통 → 몰입 에스컬레이션 | 매몰비용+자기상 이중 고착 인식 |
| CM3 | 축2→축A·C | 단일 속성→"좋은 사람"단순화→호혜·통일성 입력 조작 | 첫인상·단속성이 관계 전체 결정 |
| CM4 | 축5→축A·C | 감정이 호혜·통일성의 입구이자 증폭기(자기강화) | 감정 활성화 후 사회적 요청 |
| CM5 | 축4↔축B | 스토리텔링↔자기상. 쌍방향 자기강화 루프 | 자기 서사 변화 = 자기상 변화 = 스토리 재편집 |
| CM6 | 축6→축A | 기억 편향: 호의 과대/과소 저장, 미해결 부채감 선명 유지 | 미해결 호혜가 기억에 오래 남음 |
| CM7 | 축A↔축B | 보답 행동→"갚는 사람" 정체성→호혜 지속(자기강화) | 최초 호혜가 정체성 고착 = 외적 동기 불필요 |
| CM8 | 축C→축B | 정체성 통합("우리 하나")→자기상 확장→이탈=자기상 파괴(비가역) | 강한 소속감: 이탈 방지 효과적이나 윤리 위험 |

## 3. SDT 6 메타원리 (SM1~SM6)

| # | 접합 | 메커니즘 | 설계 함의 |
|---|---|---|---|
| SM1 | 축1→축E | 인지 고갈 → 내면화 정체 → 외적/내사 조절 고착 | 피로·스트레스: 내면화 기대 금물 |
| SM2 | 축5→축D | 프레이밍(정보적 vs 통제적) → 인과소재 지각 → 동기 질 결정 | "왜"(정보적) vs "해야"(통제적) = 자율성 보존 vs 파괴 |
| SM3 | 축B↔축E | 건강한 자기상(가치 내면화) vs 조건부 자기상(불안 기반 내사) | 조건부 자기상 = "~해야만 나는 가치" = 불안 동기 |
| SM4 | 축3↔축D | 자율성 상실 = 손실 → 반항 2.5배 증폭 | 자율성 박탈의 반발 비용 > 자율성 회복 비용 |
| SM5 | 축A→축E | 호혜 의무감 → 내면화 우회 → 외적 조절 즉시 순응 | 호혜 순응: 빠르나 비내면화. 의무감 사라지면 중단 |
| SM6 | 축D↔축E | **마스터 키**: 자율+충족=상승나선 / 통제+좌절=하강나선 | 상승/하강 나선 진입 판별이 최우선 |

## 4. 진화심리학 8 메타원리 (EM1~EM8)

| # | 접합 | 메커니즘 | 설계 함의 |
|---|---|---|---|
| EM1 | 축G→축3 | 손실 회피 = 오류 관리 특수사례. EEA 생존 경계의 비선형 비용→2.5배 | 손실 회피 강도는 진화적 교정됨. 변경 불가 |
| EM2 | 축G→축5 | 감정 선행 = 다시스템 동시 조율의 진화적 필연(S2보다 빠름) | 감정 선행은 적응, 억제하면 비용 증가 |
| EM3 | 축F→축4 | 과신 = 번식 전략(지위 획득·짝 유인). 자기기만 적응적 이점 | 지위 경쟁 맥락에서 과신 자동 증폭 |
| EM4 | 축F·G→축2 | 속성 대체 = 정보 불투명 환경의 적응. 번식가치·위협·친족 추론 | 신체적 단서(대칭·비율·나이)→자동 속성 대체 |
| EM5 | 축G↔축7 | 무임승차 탐지 = 호혜 체계의 면역 반응. 영역 특이적 자동 모듈 | 공정성 위반 탐지: 자동·즉각·강력. 설득 어려움 |
| EM6 | 축G→축C | 혐오 = 내/외집단 경계의 생물학적 기원. 행동 면역 과잉 일반화 | 혐오 기반 편견: 인식 가능하나 자동 반응 제거 어려움 |
| EM7 | 축G→축1·3·5·6 | 환경 불일치: EEA 적응 + 현대 초과 자극 → 부적응 체계적 생성 | 정보·음식·위험 신호 과잉: 기존 메커니즘 체계적 오작동 |
| EM8 | 축F→축E·축4 | 번식 전략↔지위 추구. 지위=내재적 동기로 설계 | 지위 추구: 진화적 뿌리. 내재적 동기와 구분 어려움 |

## 5. 감정이론 8 메타원리 (EmoM1~EmoM8)

| # | 접합 | 메커니즘 | 설계 함의 |
|---|---|---|---|
| EmoM1 | 축5→축H | 감정 선행→구성·조절의 전제(원재료+필요성) 설정 | 감정 활성화 필수 → 구성·조절 과정 시작 |
| EmoM2 | 축H↔축1 | 감정 조절=인지 자원 소모, 자원 고갈=조절 실패(쌍방향) | 조절 과부하→인지 능력 전반 저하. 예산 필요 |
| EmoM3 | 축H→축6 | 감정 구성→기억 편집 감정적 필터. 감정 종류=기억의 색 | 감정 구성 변화 = 같은 경험의 기억 재편집 |
| EmoM4 | 축3↔축H | 참조점→감정 구성 입력, 감정 경험→참조점 이동(쌍방향) | 손실/이득 맥락↔감정 양방향 상호 결정 |
| EmoM5 | 축H↔축E | 감정 조절→욕구충족 질 결정, 욕구충족→조절 자원 보충(쌍방향) | 조절 역량 = 동기 체계 건강도. 상승/하강 나선 분기 |
| EmoM6 | 축G→축H | 위협 관리 체계→감정 구성의 진화적 원형(공포·혐오·분노) | 기본 감정: 진화적 기원. 문화적 범주 추가 |
| EmoM7 | 축H→축7 | 감정 세분화 정밀도↔사회적 판단 차별성 | 감정 세분화 높음 = 사회적 판단 정밀·공정 |
| EmoM8 | 축H↔축D | 자율적 조절→인과소재 내부, 인과소재→조절 동기(쌍방향) | 자율적 조절 = 인과소재 내부화. 강제=외부화 |

## 6. 행동설계 8 메타원리 (BM1~BM8)

| # | 접합 | 메커니즘 | 설계 함의 |
|---|---|---|---|
| BM1 | 축I↔축1 | 습관 자동화→인지 자원 해방, 자원 고갈→습관 퇴행(쌍방향) | 좋은 습관 자동화 = 인지 예산 확보. 피로→나쁜 습관 복귀 |
| BM2 | 축5→축I | 감정: 습관 루프 트리거(부정→위안) + 보상(긍정) 동시 공급 | 감정 드라이브 습관: 형성 빠르나 의존 위험 |
| BM3 | 축I↔축B | 습관 반복→정체성 증거, 정체성→동기 비용 제거(자기강화) | "나는 ~하는 사람" = 최견고 습관 앵커 |
| BM4 | 축I↔축E | 보상→욕구충족 경로 결정, 충족→루프 내재적 에너지(쌍방향) | 진짜 욕구 충족 확인. 대체물 보상=악순환 |
| BM5 | 축I↔축D | 자율적 선택 습관→의지력 보호, 자동화→자율성 강화(쌍방향) | 강제 습관=의지력 소모. 자율적 선택=절약 |
| BM6 | 축3→축I | 손실 회피: 기존 습관 고착(바꾸면 손실) + 신규 포기(못 바꾸면 손실) | 습관 전환: 손실 프레이밍 회피. 이득으로 전환 |
| BM7 | 축H→축I | 감정 조절 실패→습관이 감정의 "비상 출구"→습관 의존 증가 | 감정 조절 강화 = 나쁜 습관 예방의 상류 개입 |
| BM8 | 축G→축I | 환경 불일치+초과 자극→습관 루프 부적응적 과잉 강화 | SNS·정크푸드·도파민 해킹: 습관 루프 과잉 강화 |

## 허브 축 3대

| 허브 | 연결 메타원리 | 핵심 역할 |
|---|---|---|
| **축H (감정)** | EmoM1~8 | 인지-감정-동기 전체 중심. 감정이 모든 축에 관여 |
| **축I (행동)** | BM1~8 | 모든 이론을 "실행"으로 통합. 습관이 인지·감정·동기·진화 연결 |
| **축E (동기)** | SM1·3·5·6, EmoM5, EM8, BM4 | 자율적 행동의 에너지원. **SM6 상승/하강 나선 분기점** |

## 자기강화 루프 6개

| 루프 | 메타원리 | 메커니즘 | 위험/기회 |
|---|---|---|---|
| 감정-기억 | M5 | 감정→기억 선별→감정 강화 | 부정 감정 고착 위험 |
| 호혜-정체성 | CM7 | 보답→자기상→호혜 지속 | 외적→내적 전환 기회 |
| 인과소재-욕구충족 | SM6 | 자율+충족=상승 / 통제+좌절=하강 | 나선 방향이 전체 동기 결정 |
| 습관-정체성 | BM3 | 습관→증거→정체성→동기비용제거 | 정체성 고착의 양면성 |
| 습관-욕구충족 | BM4 | 보상→충족경로, 충족→에너지 | 진짜 욕구 vs 대체물 분기 |
| 감정조절-욕구충족 | EmoM5 | 조절→충족질, 충족→조절자원 | 조절 실패의 하향 나선 |
