# 저활용 메타원리 활성화 가이드

> Human Skill 스포크. 44개 메타원리 중 저활용 ~34개의 분류 및 활성 트리거 조건.
> 시뮬레이션 검증에서 M5, CM7, BM3, BM4, EmoM5 등 ~10개만 반복 활성화된 문제를 해소.

---

## 3분류 체계

| 분류 | 메타원리 예시 | 활성 트리거 조건 |
|---|---|---|
| **도메인 특화형** | M2(프레이밍→S1대체), M3(가용성→두려움), CM2(초기양보→호혜), CM5(이질집단→배척), EM3(기대불일치→동기이탈) | 특정 도메인(마케팅·협상·교육 등)에서만 빈번 활성. 도메인 키워드 매칭 시 우선 탐색 |
| **조합 전용형** | M4(S2피로→S1회귀), CM3(공개선언→구속), SM3(외적보상→동기전이), EmoM3(감정명명→조절촉진), BM5(트리거제거→습관소거) | 단독 활성 드뭄. 2개+ 축 교차 시 "브릿지 원리"로 작동. `--detail`에서 조합 관계 확인 |
| **극단 상황형** | M6(감정홍수→판단마비), CM6(권위복종→자율포기), SM5(외적통제→학습무력감), EM6(분열위협→극단반응), EmoM7(감정억제→역반등), BM7(보상소거→소거폭발) | 평시에 비활성. 스트레스·위기·실패 상황에서 급활성. "극단/위기/실패" 키워드 감지 시 우선 탐색 |

## 사용법

스크리닝 후 활성 축이 3개+ → `--detail`로 메타원리 확인 시, 위 3분류 중 해당 조건 매칭 여부를 1회 체크. 매칭되면 저활용 원리도 후보에 포함.
