# SDT 고유 2축 상세 — 내부 인과 체인 + 설계 원칙

## 고유축D. 자율성 욕구와 인과소재 역학

**축 자명명제**: 인간은 행동의 기원이 자기 내부에 있다는 지각을 본질적으로 추구하며, 이 지각의 충족·훼손이 동기·안녕감·행동의 질을 결정한다

### 체인 D-① 외적 보상 → 인과소재 이동 → 내재적 동기 소멸

```
유형 보상은 행동과 외부 유인 사이에 명시적 연결을 만든다
  → 사전 예상된 보상은 행동 개시 전에 외부 유인을 인식하게 한다
    → "보상 때문에 한다"는 자기 해석이 인과소재를 외부로 이동
      → 인과소재 외부 이동 → 자율성 훼손 → 내재적 동기 약화
        → 자기 해석은 보상 제거 후에도 잔존
          → 원래의 내적 인과소재 복귀 차단 (비가역성)
```

**의미**: 외적 보상은 인과소재를 외부로 이동시키고, 이 이동은 비가역적. 과잉정당화 효과와 동기 밀어내기의 핵심 메커니즘.

### 체인 D-② 자율성 지지 환경 → 인과소재 내부 유지 → 동기 촉진

```
선택은 행동의 기원을 자기 내부에 위치시킨다
  → 이유 설명은 행동과 개인적 가치 사이의 연결을 인식하게 한다
    → 가능 언어는 행동의 기원을 자기 선택으로 프레이밍
      → "내가 원해서"(내적 인과소재) = 자율적 동기의 핵심 경험
        → 내적 인과소재 → 자기 효력감 생성
          → 효력감 → 탐색·시도 촉진 → 흥미·창의성 증가
```

**의미**: 자율성 지지 환경의 3요소(선택·이유·비통제적 언어)는 모두 인과소재를 내부에 유지시키는 메커니즘으로 수렴.

### 체인 D-③ 자율성 훼손 → 양극 반응 (복종 vs 반항)

```
외적 인과소재는 자기 효력감을 소멸시킨다
  → 효력감 소멸 → 무력감
    → 자기 의지 부정 → 소속 맥락과 심리적 거리 증가(소외)
      → 양극 분기:
          경로1: 에너지 절약 → 수동적 복종
          경로2: 자율성 회복 시도 → 반항·저항
```

**의미**: 자율성 훼손에 대한 반응은 단일하지 않다. 에너지 수준에 따라 복종(저에너지)과 반항(고에너지)으로 분기.

### 설계 원칙

- 외적 보상의 치명성: 내재동기 파괴가 비가역적
- 자율성 지지 3요소: 선택권 + 근거 설명 + 감정 인정
- 통제 회피: 순종 또는 반항만 초래 (둘 다 자율적 동기 파괴)
- SM2 연동: 프레이밍(정보적 vs 통제적)이 인과소재 지각을 결정 → 동기의 질 결정
- SM4 연동: 자율성 상실은 손실 회피 활성화 → 반항 강도 증폭 (2.5배 비대칭)

## 고유축E. 내재적 동기와 욕구 충족 역학

**축 자명명제**: 인간은 3대 보편 욕구를 가지며, 이 욕구의 충족·좌절이 동기의 내면화 수준, 행동의 질, 안녕감을 체계적으로 결정한다

### 체인 E-① 욕구 충족 → 내면화 → 자율적 동기 → 최적 기능 (밝은 경로)

```
성장에는 심리적 영양소(3대 욕구 충족)가 필요하다
  → 자율성 지지 + 이유 설명 + 감정 수용 + 관계적 안전감 → 내면화 촉진
    → 내면화 진행 → 에너지원이 자기 체계에 접근
      → 가치 인식(확인) → 정체성 일치(통합)
        → 활동 자체의 흥미·즐거움(내재적 동기)
          → 온전한 집중 → 심층 처리·몰입 → 최적 성과
```

**의미**: 욕구 충족→내면화→자율적 동기→최적 기능의 선순환 경로. SDT의 "밝은 경로".

### 체인 E-② 욕구 좌절 → 보상적 행동 → 악순환 (어두운 경로)

```
능동적 훼손(좌절)은 욕구 충족 경로를 파괴한다
  → 욕구 좌절 → 결핍 경험(심리적 고통)
    → 고통 완화를 위해 즉각적 보상·자극 추구(보상적 행동)
      → 욕구별 고유 보상: 반항(자율), 물질주의(유능), SNS 과몰입(관계)
        → 보상적 행동은 근본 욕구를 충족하지 못한다
          → 고통 재유발 → 보상적 행동 반복(악순환)
            → 반복 → 강화 루프 → 중독성 행동
```

**의미**: 욕구 좌절→대리만족→근본 미충족→반복의 악순환이 중독성 행동의 구조적 기원. "어두운 경로".

### 체인 E-③ 충족-좌절 비대칭과 경로 분기

```
충족 부재(결핍) = 성장 정체
  ≠ 능동적 훼손(좌절) = 경로 파괴 + 방어 활성
좌절은 결핍보다 부정적 결과 강도가 크다
  → 분기:
      결핍 경로: 무동기·성장 정체
      좌절 경로: 방어·보상·병리
```

**의미**: "욕구를 안 채워주는 것"과 "욕구를 짓밟는 것"은 별도 경로. 설계에서 충족과 좌절 방지를 분리 접근해야 하는 이유.

### 내면화 연속체

| 단계 | 동기 유형 | 인과소재 | 에너지 | 지속성 |
|------|---------|---------|--------|--------|
| 무동기 | 없음 | 비인격적 | 0 | 0 |
| 외적 조절 | 보상·처벌 | 외부 | 낮음 | 보상 의존 |
| 내사 | 자아 위협·죄책감 | 부분 외부 | 중간(불안정) | 위협 의존 |
| 동일시 | 가치 인식 | 부분 내부 | 높음 | 가치 의존 |
| 통합 | 정체성 일치 | 내부 | 최고 | 자기유지 |
| 내재적 | 흥미·즐거움 | 완전 내부 | 최고 | 자기유지 |

### 설계 원칙

- 3대 욕구 동시 충족: 자율 + 유능 + 관계 (한 욕구 좌절→전체 체계 손상)
- 내면화 깊이가 동기의 질 결정
- 충족 설계와 좌절 방지를 분리 접근
- SM6 연동: 인과소재(D)가 욕구 충족의 질 필터, 욕구 충족(E)이 인과소재 유지 에너지 공급
- SM3 연동: 건강한 자기상→통합 촉진 / 조건부 자기상→내사 트랩

## SM6 핵심 역학 — D↔E 쌍방향 강화 루프

```
         인과소재 내부 유지 (축D)
                 ↓
         (외적보상 차단 ↔ 자율성 지지)
                 ↓
    3대 욕구 충족 (축E)
    ├─ 자율: 선택·결정권
    ├─ 유능: 역량 성장·달성
    └─ 관계: 소속감·신뢰
                 ↓
      상승나선 ↔ 하강나선
     (내면화 심화)  (욕구좌절)
```

| 상황 | 인과소재 | 욕구 상태 | 결과 |
|------|---------|---------|------|
| 외적 보상 제공 | 외부로 이동 | 어떤 상태든 | 내재동기 소멸 |
| 자율성 지지 + 충족 | 내부 유지 | 3대 욕구 충족 | 상승나선 |
| 통제 + 조건부 보상 | 외부 이동 | 욕구 좌절 | 하강나선 |
| 자율성 지지 | 내부 유지 | 욕구 좌절 | 정체(개선 가능) |
```
