# 감정이론 고유축H 상세 — 내부 인과 체인 + 설계 원칙

## 고유축H. 감정 구성과 조절 역학

**축 자명명제**: 인간의 감정은 예측적으로 구성되고, 시점 의존적으로 조절되며, 개념적 세분화에 의해 정밀도가 결정된다

---

### 체인 H-① 신체 예산 → 예측적 구성 → 감정 인스턴스 생성

```
인간의 뇌는 내부 신체 상태를 지속적으로 모니터링한다(내수용 감각)
  → 내수용 감각이 쾌-불쾌 × 각성의 2차원 정동(core affect)을 생성한다
    → 뇌의 핵심 기능은 신체 자원의 예측적 조절(allostasis)이다
      → 뇌는 과거 경험을 바탕으로 현재 신체 상태의 '의미'를 예측한다
        → 예측에 감정 개념이 적용되면 특정 감정 인스턴스가 구성된다
          → 동일 신체 상태라도 다른 개념이 적용되면 다른 감정이 된다
```

**의미**: 감정 생성은 수동적 반응이 아니라 능동적 구성. 신체 예산 변동→정동→예측→범주화의 4단계 파이프라인에서, 감정 개념이 결정적 분기점. 개념이 없으면 정동은 모호한 불편함으로 남는다.

---

### 체인 H-② 감정 세분화 → 조절 정밀도 → 행동 적응성

```
양육자의 감정 명명이 아동의 감정 개념 레퍼토리를 형성한다
  → 문화의 감정 어휘가 가용한 감정 범주의 폭을 결정한다
    → 풍부한 감정 개념 = 동일 정동을 더 세밀하게 범주화(세분화)
      → 세분화된 범주 = 각 감정에 맞는 조절 전략을 정밀하게 선택 가능
        → 정밀한 조절 = 충동적 행동(과음·공격) 감소
          → 세분화↑ → 조절력↑ → 행동 적응성↑
```

**의미**: 감정 세분화는 감정 조절의 인프라. "명명할 수 있으면 길들일 수 있다"는 것은 개념이 조절 전략 선택의 전제 조건이기 때문. 양육·교육·문화에 의해 변형 가능한 메커니즘.

| 세분화 수준 | 감정 표현 | 조절 전략 | 행동 결과 |
|------------|---------|---------|---------|
| 매우 높음 | "기준 부족감" + "성취동기" + "경쟁 불안" | 목표 세분화·마일스톤 설정 | 명확한 액션 |
| 높음 | "슬픔" vs "좌절" vs "외로움" | 각각 공감 vs 도전 vs 연결 | 맞춤형 대응 |
| 중간 | "기분 나쁨" | 일반적 대처(산책, 음악) | 시행착오 |
| 낮음 | "뭔가 이상한데?" | 회피·억제 | 부적응 강화 |

---

### 체인 H-③ 조절 시점 → 비용 비대칭 → 유연성이 역량

```
감정 생성은 상황→주의→평가→반응의 시간 경과를 가진다
  → 각 시점에서 서로 다른 조절 전략이 개입 가능하다
    → 초기(선행 초점) 개입은 감정 생성 전에 의미를 바꾸므로 비용이 낮다
    → 후기(반응 초점) 개입은 이미 생성된 감정을 억제하므로 비용이 높다
      → 재평가: 의미 1회 재구성 → 감정 경험·각성 모두 감소
      → 억제: 표현만 차단, 감정 경험 불변 + 각성 역설적 증가
        → 단일 전략 고수는 맥락 부적합 시 실패
          → 맥락에 따른 유연한 전략 전환 능력 = 조절 역량의 핵심
```

**의미**: 감정 조절의 "언제"가 "무엇을"보다 중요. 동일한 부정 감정도 초기 개입(재평가)과 후기 개입(억제)에서 정반대 결과를 초래. 단일 전략의 능숙함보다 전략 전환의 유연성이 적응적.

| 시점 | 전략 | 인지비용 | 정서효과 | 추천 상황 |
|------|------|---------|---------|---------|
| 사전(선택) | 상황회피 | 매우낮음 | 높음 | 반복적 스트레스 |
| 사전(수정) | 상황개선 | 낮음 | 높음 | 개선 가능 환경 |
| 진행중 | 주의이동 | 중간 | 중간~높음 | 주의산만 가능할 때 |
| 진행중 | 인지재평가 | 높음 | 중간~높음 | 의미 재구성 필요 |
| 사후 | 반응억제 | 매우높음 | 낮음 | 긴급상황 회피용만 |

## 허브 축 특성 — 8개 메타원리 접합

고유축H는 7개 축과 직접 교차하는 **허브 축**이다.

| 메타원리 | 접합 | 방향 | 핵심 |
|---------|------|------|------|
| EmoM1 | 축5→축H | 단방향 | 감정 선행이 구성·조절의 전제 조건(원재료+필요성) 설정 |
| EmoM2 | 축H↔축1 | 쌍방향 | 감정 조절이 인지 자원을 소모하고, 자원 고갈이 조절 실패를 유발 |
| EmoM3 | 축H→축6 | 단방향 | 감정 구성이 기억 편집의 감정적 필터를 결정 |
| EmoM4 | 축3↔축H | 쌍방향 | 참조점이 감정 구성의 입력이 되고, 감정 경험이 참조점을 이동시킨다 |
| EmoM5 | 축H↔축E | 쌍방향 | 감정 조절이 욕구 충족의 질을 결정, 욕구 충족이 조절 자원을 보충 |
| EmoM6 | 축G→축H | 단방향 | 위협 관리 체계가 감정 구성의 진화적 원형(공포·혐오·분노) 제공 |
| EmoM7 | 축H→축7 | 단방향 | 감정 구성의 정밀도가 사회적 판단의 차별성을 결정 |
| EmoM8 | 축H↔축D | 쌍방향 | 감정 조절의 자율성 경험이 인과소재를 결정, 인과소재가 조절 동기를 결정 |

## 설계 원칙

### 설계 가능/불가능 영역

| 영역 | 가능성 | 전략 |
|-----|--------|------|
| 신체신호 제어 | 불가 (자율신경) | 억지 통제 비추 |
| 감정 범주화 인식 | 가능 | 맥락·예측 재구성 |
| 세분화 발전 | 가능 | 어휘·훈련으로 발달 |
| 조절 시점 선택 | 가능 | 의도적 전략 훈련 (가장 효과적) |
| 억제 의존 탈출 | 가능 | 다른 전략 습득 (정신건강 필수) |

### 감정 인식→세분화→조절 파이프라인

```
1. 신체신호 인식 → "심박·호흡·긴장 감지했나?"
2. 맥락 확인 → "지금 어디? 누가 있나?"
3. 예측 신호 검토 → "이 반응이 자연스러운가?"
4. 감정 범주화 → "정확한 라벨은? 혼합 감정은?"
5. 조절 전략 선택 → 상황선택 > 수정 > 주의이동 > 재평가 > 억제
6. 행동 실행 & 피드백 → "효과 있었나? 다음번에는?"
```
